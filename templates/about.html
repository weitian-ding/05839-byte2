Where did this data come from?

The dataset behind the FusionTable is a wine review dataset published on Kaggle. While the orginal dataset has more fields like the winery which produced the wine, we are only interested in a subset of those fields (see the FusionTable snippet). The publisher of the dataset says that the data is scraped from WineEnthusiast on November 22nd, 2017. There had been a few previous iterations of the dataset. I downloaded the second release of the dataset and uploaded the csv to a Google FusionTable, then I picked columns of interest and hided the rest of the columns. The publisher also shared the source code for the scrapper.

Who has access to it?

Since the dataset is hosted on Kaggle, anyone on the public Internet has access to it.
