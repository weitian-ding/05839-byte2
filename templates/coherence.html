Is the data coherent?

The dataset is coherent, we can look at some statistics for each column. Wine price and rating are two numeric fields in the dataset (see table 2). Wine price is highly skewed (see figure 8), while wine ratings follow a normal distribution (see figure 9, here we see wine rating as a continuous random variable). We see that there are high-end wines which are priced over $3,000 (see table 2), but this is still within a reasonable price range. We can also examine wine variety.

We count the number of reviews for each wine variety in the dataset (see table 10), and we find that there are few wine varieties which contribute to the bulk of wine reviews. We can illustrate this by plotting the number of wine reviews for each variety in sorted by the number of wine reviews (see figure 10), we see a long-tailed distribution. We can also do the same to the country where a wine is produced (see table *, figure 11), and we find that most wine reviews are for American, French and Italian wine.

Next, we look at the actual text of wine reviews. Using word cloud, we see the frequently appearing word n-gram in the wine review corpus are “black cherry”, “drink now”, “fruit flavor” and “full-bodied” (see figure 12), which makes sense since they are all topically related to wine. We can also look at the length of wine reviews (see figure 13). For each wine rating bucket, we plot a box plot of wine review length. Interestingly, we find that positive wine reviews are generally longer than less positive ones (see figure 14).
